Csysfetch is a minimal command line system info fetcher. Csysfetch is fully supported to run on Linux, and there is work being done to bring support to macOS. Csysfetch is written in C primarily but calls upon bash commands and a bash script for it's functionality.


<img src="./pics/csysfetchgrn.png" align="center" height="240px" width="550">


FEATURES
--------------------
* Full ANSI color code support
* Easily configurable color system


<img src="./pics/csysfetchpurp.png" align="center" height="240px" width="550">




DEPENDENCIES
--------------------
 * git
 * bash 5.2+
 * make
 * gcc


INSTALL GUIDE
--------------------


First clone the repository:

    git clone https://codeberg.org/MrNone1/csysfetch.git

Then run the following commands to compile cysfetch:

    cd csysfetch
    make install

To run csysfetch you can use either of the following commands:

    make run
or

    ./csysfetch
